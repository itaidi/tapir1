Тестовое задание на позицию «Backend-разработчик» 

Написать API сервис витрины автомобилей. 

1 Необходимо экспортировать данные из внешних источников:
    - http://static.tapir.ws/new_cars.json - новые авто
    - http://static.tapir.ws/used_cars.xml - авто с пробегом
импорт должен происходить каждый час.

2 Написать API фильтра авто, фильтровать можно по всем параметрам авто, параметры объединяются (and). Например:
запрос curl http://localhost/api/stock?year_from=2000&year_to=2010&price_less=1000000 найдет авто выпущенные с 2000 по 2010 и ценой меньше миллиона. Должна быть пагинация.

3 Написать документацию к API в swagger. Должна быть возможность сгенерировать документацию в проекте одной командой redoc-cli.

4 Проект должен находится в открытом git репозитории, запуск локально через docker. То есть проверяться работа будет так - git clone & docker-compose up. Описать в README.md порядок запуска проекта.
